In the End by Linkin Park



Hello! This website is presenting a word cloud created from the results of an assignment for my Data Strucures class in Brooklyn College.
This particular word cloud reflects the lyrics of the song "In the End" by
Linkin Park. The darker the word, the more frequently it appears in the song!

  • This word cloud was created by:
  • -A java program I made called "Word Count"
  • -Word Count read in song lyrics from an input file and printed a word frequency list into an output file
  • -This cloud was the result of that list
